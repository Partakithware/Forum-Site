## Deployment Instructions

### Deploy Backend to Railway

1. Create account at https://railway.app
2. Install Railway CLI:
   ```bash
   npm install -g @railway/cli
   ```

3. Login and initialize:
   ```bash
   railway login
   cd backend
   railway init
   ```

4. Set environment variables in Railway dashboard:
   - `SESSION_SECRET`: Generate a random string
   - `FRONTEND_URL`: Your frontend URL (e.g., `https://your-site.web.app`)
   - `NODE_ENV`: `production`

5. Deploy:
   ```bash
   railway up
   ```

6. Get your backend URL from Railway dashboard (e.g., `https://your-app.railway.app`)

### Deploy Frontend to Firebase

1. Install Firebase CLI:
   ```bash
   npm install -g firebase-tools
   ```

2. Login and initialize:
   ```bash
   firebase login
   cd frontend
   firebase init hosting
   ```

3. Update `config.js` with your Railway backend URL:
   ```javascript
   const API_URL = 'https://your-app.railway.app/api';
   ```

4. Deploy:
   ```bash
   firebase deploy
   ```

### Alternative: Deploy Frontend to Netlify/Vercel

**Netlify:**
```bash
npm install -g netlify-cli
cd frontend
netlify deploy --prod
```

**Vercel:**
```bash
npm install -g vercel
cd frontend
vercel --prod
```

## Local Development

1. **Start Backend:**
   ```bash
   cd backend
   npm install
   npm run dev
   ```

2. **Start Frontend** (use any local server):
   ```bash
   cd frontend
   python -m http.server 8080
   # or
   npx serve .
   # or
   npx live-server
   ```

3. Open `http://localhost:8080` in your browser
